# LcodeDO（LDStatus）

> 一个为 [Linux.do](https://linux.do) 与 [IDCFlare](https://idcflare.com)（基于 Discourse 的社区）打造的油猴用户脚本，在你的浏览器侧边显示**信任级别进度**、**TL3 100 天进度**、**阅读时间统计与等级**、**排行榜**、**自动阅读**等增强能力，并支持跨浏览器 / 跨设备的**云同步**。

- 脚本名称：`LcodeDO`
- 当前版本：`0.0.7`
- 作者：Lcode
- 借鉴二开于：[JackLiii](https://github.com/JackLiii)
- 许可证：MIT
- 运行环境：Tampermonkey / Violentmonkey 等用户脚本管理器
- 匹配站点：`linux.do`、`idcflare.com`，以及 `cdk.linux.do`、`credit.linux.do`（数据桥接）

---

## ✨ 功能特性

### 1. 信任级别进度
抓取用户在社区中的关键指标（浏览话题、已读帖子、获赞、送出赞、回复、停留天数等），实时计算并可视化展示距离下一信任级别（TL0 → TL4）的进度条。

### 2. TL3 / 三级用户 100 天进度
面板新增「TL3」tab，用于查看信任等级 3（活跃用户）的晋级 / 保级状态：

- 按过去 **100 天滚动窗口** 汇总 TL3 核心指标。
- 自动区分当前用户是「晋级模式」还是「保级模式」。
- 展示总体完成度、已达标项目数、待处理项目数。
- 每个指标都有独立进度条与缺口提示，例如「还差多少已读帖子 / 送出赞」。
- 对举报、禁言、封禁等反向限制项进行风险提示。
- 优先使用 Connect 页面提供的真实 100 天数据；Connect 不可用时可回退到云端同步或 summary 累计统计，并明确标注「仅供参考」。
- 内置 TL3 准则说明，可在进度页一键查看完整规则。

### 3. 历史趋势
周期性记录指标快照，绘制历史趋势，让你看到自己「水帖成长曲线」随时间的变化。

### 4. 里程碑通知
对关键指标设置了里程碑阈值，达成时弹出庆祝通知：

| 指标 | 里程碑 |
| --- | --- |
| 浏览话题 | 100 / 500 / 1000 / 2000 / 5000 |
| 已读帖子 | 500 / 1000 / 5000 / 10000 / 20000 |
| 获赞 | 10 / 50 / 100 / 500 / 1000 |
| 送出赞 | 50 / 100 / 500 / 1000 / 2000 |
| 回复 | 10 / 50 / 100 / 500 / 1000 |

### 5. 阅读时间统计与等级系统
- 通过监听鼠标、键盘、滚动、可见性、窗口焦点等事件，精准统计真实阅读时长；切到后台 / 失焦自动暂停，避免「挂机刷时长」误差。
- 多标签页安全：同一账号只允许一个「领导者」标签页写入数据，避免重复计数。
- **10 级阅读等级**（🌱 → 📖 → 📚 → 🔥 → ⚡ → 🏆 → 👑 → 💎 → 🌟 → 🚀），阈值可由服务端动态下发，本地缓存 24 小时。
- 支持年度数据聚合与历史数据迁移。

### 6. 排行榜系统
- 登录后可加入阅读排行榜，与其它用户比拼阅读时长，展示前若干名及「我的排名」。
- 手动刷新有 5 分钟冷却，自动同步周期 15 分钟。

### 7. 我的活动
集中查看自己在社区中的活动数据（话题、回复、点赞等）。

### 8. 自动阅读
- 在话题列表页点击标题栏的 📖 按钮即可启动，自上而下依次打开文章并停留指定时长（自动滚动浏览），模拟真实阅读以累计阅读时长 / 已读帖子 / 浏览话题。
- 可配置：每篇停留时长（预设 1/2/3/4/5 分钟或 10/20/30 秒）、是否自动滚动、单次浏览篇数上限（0 = 不限）。
- 运行状态持久化到 GM 存储，可跨页面跳转继续；运行时右下角悬浮 HUD 显示进度。
- 单次最多排队 200 篇。

### 9. 帖子标题悬浮预览
鼠标悬停话题标题即弹出预览卡片，展示帖子概要信息，无需点进去即可判断是否值得读。

### 10. 云同步
- 登录后阅读数据自动上传 / 下载到云端，支持跨浏览器、跨设备同步。
- 上传周期 30 分钟，下载周期 6 小时，状态检查 10 分钟，并设有同步窗口避免冲突。
- 身份切换（更换账号）时自动清理旧 token，防止数据串号。

### 11. CDK / LDC 桥接
当脚本运行在 `cdk.linux.do` 或 `credit.linux.do` 时，会切换为「数据桥接」模式：用同域原生 `fetch`（自动带 Cookie）请求接口，再通过 `postMessage` 把结果回传给主站页面，绕过跨域限制。

### 12. OAuth 登录
脚本启动时尽早捕获 URL hash 中的 OAuth 回调结果，避免被 Discourse 路由处理掉而丢失登录态。

### 13. 阅读沙漏计时器
面板折叠后即变为一个沙漏计时器（替换原 logo），显示**今日阅读时长**（持久化，刷新不归零），可拖拽。正在记录时沙漏呼吸动画、沙流闪烁；失焦/切后台自动暂停。点击沙漏展开面板并切换到「阅读」tab。

### 14. 阅读 tab：每日目标环 + 仪表盘
面板新增「阅读」tab（顺序：要求 → 阅读 → TL3 → 历史），内含 SVG 进度环展示今日阅读时长相对每日目标（可在「设置 → 🎯 阅读目标」调整）的完成百分比。达标当天沙漏翻转动画 + `GM_notification` 庆祝，每天仅通知一次。tab 内还展示今日/累计阅读、已读帖数、阅读等级、本次会话、阅读速度与防沉迷设置，打开时实时刷新。

### 15. 阅读速度统计
「阅读」tab 内结合「累计阅读时长」与「已读帖子数」（来自 `HistoryManager` 历史）计算平均每篇用时与每小时阅读帖数，可反向校准自动阅读的停留时长建议。

### 16. 防沉迷提醒
连续阅读超过设定分钟数（默认 45 分钟，「阅读」tab 内可开关并调整）后，温柔推送「起来走走」通知，每 30 分钟最多提醒一次，避免打扰。

### 17. 已读帖子标记
话题列表页自动给已访问过的话题加上「已读」徽标并降低不透明度。已读话题 id 按用户隔离存储（GM，上限 2000 条），跨页面持久。

### 18. 继续阅读（断点续读）
在话题页自动记录离开时的话题与滚动位置；返回列表页时顶部出现「↩ 接着读：xxx」按钮，点击后恢复到上次离开的大致位置（7 天内有效）。

## 📦 安装

1. 安装用户脚本管理器浏览器扩展：[Tampermonkey](https://www.tampermonkey.net/)（推荐）或 Violentmonkey。
2. 打开扩展管理面板 → 新建脚本 → 粘贴 `LDStatus.js` 全部内容并保存；或直接将 `.js` 文件拖入扩展。
3. 访问 [linux.do](https://linux.do) 或 [idcflare.com](https://idcflare.com)，页面右侧即出现 LcodeDO 浮动面板。

> 不登录也可正常使用本地功能（阅读计时、悬浮预览等）；登录后才会启用云同步与排行榜。

## ⚙️ 可配置项

通过面板设置入口与 GM 存储管理：

| 配置 | 说明 |
| --- | --- |
| 自动阅读每篇停留时长 | 预设 1/2/3/4/5 分钟 或 10/20/30 秒 |
| 自动阅读自动滚动 | 开 / 关 |
| 自动阅读篇数上限 | 0 = 不限 |
| 防沉迷提醒 | 开 / 关，连续阅读分钟阈值（默认 45） |
| 官网入口 URL | 每日动态更新（`ldsp_website_url`） |
| 西瓜字体大小 | `ldsp_melon_font_size` |
| 桥接调试日志 | `ldsp_debug_bridge`（默认关闭） |

## 🗂 项目结构

```
LDStatus.js   # 单文件用户脚本（含全部功能与内联样式）
README.md     # 本文档
```

脚本为单文件架构，主要模块（按代码顺序）：

- 跨域白名单与 CDK/LDC 桥接页
- OAuth 登录结果捕获
- 常量与配置（里程碑、阅读等级预设、各定时器周期）
- `ReadingTracker` — 阅读时间追踪器
- `LeaderboardManager` — 排行榜管理器
- `CloudSyncManager` — 云同步管理器
- `AutoReader` — 自动阅读
- `ReadingHUD` — 阅读沙漏计时器 + 阅读仪表盘（目标环 / 速度 / 防沉迷）
- `Panel._renderTL3Progress` — TL3 100 天晋级 / 保级进度页
- `TopicListEnhancer` — 已读帖子标记 + 继续阅读断点续读
- `ReadHistoryStore` — 已读话题 / 断点 / 防沉迷配置存储
- 面板 UI、悬浮预览、我的活动
